# 🎬 ComicFlow · 漫剧创作工作流平台

AI驱动的漫剧创作流水线管理平台。

## 核心功能

### 📜 剧本解析
上传剧本 → AI自动提取角色、场景、道具，自动生成资产包

### 👤 角色管理
- 支持角色多版本管理（不同时期/服装/武器/配饰）
- 每个版本独立生成图片
- 角色与分镜关联

### 🏞️ 场景管理
- 场景类型标签（室内/室外/幻想/历史）
- 时间设定（白天/夜晚/黎明/黄昏）
- 场景与分镜关联

### 🎒 道具管理
- 道具分类（武器/服装/配饰/工具）
- 道具与分镜关联

### 🎞️ 分镜制作
- AI自动从剧本生成分镜
- 关联资产：自动带入角色+场景+道具+配饰
- 支持批量生成提示词

### 🤖 AI生图集成
支持主流AI生图工具提示词生成：
- Flux (1.1 Pro / Dev / Schnell)
- Midjourney v6
- DALL·E 3
